Тема КП: Программа упаковщик исполняемых файлов формата exe. (Packager)
Утилита представляет собой консольное приложение, которое сжимает бинарный код exe файла и приклепляет рядом с ним код распоковщика. При запуске сжатого exe файла распаковщик распаковывает бинарный код в память и запускает его.

Возможно добавление шифрования.
